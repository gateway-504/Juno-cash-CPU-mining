##  Mining

### Download the binary

```
wget https://github.com/juno-cash/junocash/releases/download/v0.9.4/junocash-0.9.4-linux64.tar.gz
```

### Extract

```
tar -xvf junocash-0.9.4-linux64.tar.gz  
```

### Go into the extracted folder (adjust if the folder name differs)

```
cd junocash-0.9.4 
```

### Make sure binary is executable

```
chmod +x bin/junocashd
```

### Run the daemon and start mining

```
./bin/junocashd
```

When the terminal UI is up:

- Press M

- Press T

- Type -1 for max threads or any number of threads you wish to run and press Enter to start mining as per the tweetâ€™s instructions.
